# NerveAI 🚀
AI-Powered Mock Interview &amp; Virtual Internship Platform for SDE Preparation.

## ✨ Features
- AI Mock Interview System
- Real-time feedback
- Virtual Internship (5 projects)
- Certificate generation

## 🛠 Tech Stack
- React + TypeScript
- Node.js
- PostgreSQL
- AI APIs

## 📌 Modules
- Authentication
- Interview Engine
- Project Tracker
- Certificate System
