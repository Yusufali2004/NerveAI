# 📝 Contributor Activity Logs

This folder is for contributors to document their journey and technical notes. 
Please create a file named `your-github-username.md` in this folder.

### Format for your file:
## [Name] - @[github-username]
- **Issue Worked On:** #IssueNumber
- **Date:** YYYY-MM-DD
- **What I did:** Brief technical summary of the contribution.
- **Challenges:** Any hurdles you faced.
